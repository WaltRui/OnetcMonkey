# OnetcMonkey
玩客猴辅助工具，包含全网数据查询、猴子信息查询、掘金计算器、生育计算器、投食计算器、自动喂养、自动修炼等功能

软件网址：http://www.onetc.info/wkmonkey/ruan-jian-yi-jing-kai-yuan-huan-ying-start
